# Sentosa Trade Portal — Prompt Engineering Test

This repo is a test project for building a B2B ticketing platform for Sentosa Development Corporation using Claude Code. The platform is designed for tour agents to browse and purchase attraction tickets on behalf of their clients.

## prompts/

Contains the prompts used to generate this project:

- [original.md](prompts/original.md) — the initial rough prompt describing the product idea
- [upgraded.md](prompts/upgraded.md) — the upgraded A+ prompt, rewritten with full product requirements, user flows, technical specs, and visual language for Claude Code to build from

## Stack

- React + TypeScript
- Vite
- Tailwind CSS
